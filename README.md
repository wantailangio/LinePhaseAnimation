# ライン・パターンエディタ（LinePhaseAnimation）

グリッド上に直線を描き、その線分上を動く点のアニメーションを作成できる、ブラウザ完結型のパターンエディタです。位相（フェーズ）や速度・遅延などを線グループごとに設定して、波のように動くパターンを手軽に作れます。

すべて 1 枚の `index.html`（HTML / CSS / JavaScript）で完結しており、ビルドや外部ライブラリ、サーバーは不要です。PC・スマホ・タブレットのタッチ操作に対応しています。

## 主な機能

### 作画モード
- **描画モード** — グリッド上の 2 点をクリック／タップして直線を引く
- **配置モード** — 保存したグループの直線を、グリッド点へ繰り返し配置（重複可）
- **矩形敷詰モード** — ドラッグした矩形範囲内のグリッド点に直線を一括で敷き詰め。並び方向・開始隅・間引き（飛ばす数）・先頭オフセットを指定可能

### グループごとのアニメーション設定
| 項目 | 内容 |
|------|------|
| 点の速度 | 線上を動く点の速さ（px/s） |
| 角度(加算) | 配置時の回転角オフセット（°） |
| 開始遅延 | 動き出すまでの遅延（s） |
| 開始位置 | 線上の初期位置（%）→ 位相のずらしに利用 |
| 停止間隔／停止秒数 | 周期的に一時停止する間隔と長さ（s） |
| 点の数 | 1 本の線上に並べる点の数（個） |
| 点の開始ずらし | 複数点どうしの位相オフセット（%） |
| 点のサイズ | 描画する点の半径（px） |

### その他
- 🎲 **ランダム生成**（シード指定で再現可能）
- ズーム／パン（ホイール・ピンチ・中ドラッグ）
- 直線表示／グリッド表示のトグル
- **JSON 書出・読込** によるパターンの保存・共有
- プリセット読込
- グループの複数選択と設定の一括適用
- 再生 / リセット、FPS・配置数・動く点数などの統計表示

## 使い方

1. `index.html` をブラウザで開く（ダブルクリックでも可）。
2. **描画モード**でグリッド上の 2 点をクリックして直線を引く。
3. 「選択した直線をグループ保存」でグループ化する。
4. 上部の設定バーで速度・位相などを調整する。
5. **配置／矩形敷詰モード**で線を並べる。
6. **▶ 再生**でアニメーションを確認する。
7. 必要に応じて **JSON 書出**で保存する。

> ローカルファイルとして開くだけで動作します。サーバーは不要です。

## 動作環境

モダンブラウザ（Chrome / Edge / Firefox / Safari）。スマートフォン・タブレットのタッチ操作にも対応。
